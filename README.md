# Парсер медицинских документов (анализы крови)

Курсовая работа

---

## Описание проекта

Разработка приложения для автоматического извлечения медицинских показателей из PDF-документов (анализов крови), сохранения данных в облачную базу данных Supabase и визуализации динамики показателей в Metabase.

---

## Технологии

| Технология | Назначение |
|------------|-----------|
| Java 17 | Язык программирования |
| Apache PDFBox | Извлечение текста из PDF |
| Maven | Управление зависимостями |
| Supabase | Облачная база данных (PostgreSQL) |
| Metabase | Визуализация данных |
| Git / GitHub | Контроль версий |

---

## Структура проекта

| Файл / Папка | Описание |
|--------------|----------|
| `pdfs/` | Исходные PDF-файлы с анализами крови |
| `src/main/java/com/labparser/` | Исходный код Java |
| `PdfReaderTest.java` | Извлечение и очистка текста из PDF |
| `LabReportParser.java` | Парсинг медицинских показателей |
| `cleaned_*.txt` | Очищенный текст из PDF |
| `parsed_*.json` | Результат парсинга в формате JSON |
| `Анализ структуры PDF.md` | Документ с анализом структуры PDF |
| `database_info.txt` | Данные для подключения к Supabase (не в Git) |
| `pom.xml` | Maven-конфигурация проекта |
| `.gitignore` | Список игнорируемых файлов |
| `README.md` | Документация проекта |

## Основные классы

### PdfReaderTest.java

- Извлечение текста из PDF с помощью PDFBox
- Удаление строк с точками-тире
- Объединение разорванных строк
- Исправление степеней (10^9, 10^12)
- Добавление пробелов между числами и единицами измерения

### LabReportParser.java

- Чтение очищенного текста
- Поиск показателей по словарю (90+ показателей)
- Извлечение числовых значений и единиц измерения
- Сохранение результатов в JSON
- Автоматическая обработка всех cleaned_*.txt файлов
