# Анализ структуры PDF-отчётов

## Общая информация
- **Источник:** ООО "МедЭксперт" (г. Воронеж)
- **Формат:** Текстовый PDF (текст извлекается программно)
- **Структура:** Отчёт состоит из 2-3 страниц

## Структура данных

### Информация о пациенте
| Поле | Формат | Пример |

| Дата анализа | ДД.ММ.ГГГГ | 01.05.2024 |

| Дата рождения | ДД.ММ.ГГ | 31.12.90 |

### Показатели крови

#### Клиника крови
| Показатель | Единица измерения | Формат значения |
|------------|-------------------|-----------------|
| Гемоглобин (HGB) | грамм/литр | Целое число |
| Гематокрит (HCT) | % | Десятичное с запятой |
| Эритроциты (RBC) | 10^12/литр | Десятичное с запятой |
| Тромбоциты (PLT) | 10^9/литр | Целое число |
| Лейкоциты (WBC) | 10^9/литр | Десятичное с запятой |

#### Биохимия
| Показатель | Единица измерения | Формат значения |
|------------|-------------------|-----------------|
| Глюкоза | ммоль/литр | Десятичное с запятой |
| Креатинин | ммоль/литр | Целое число |
| Билирубин общий | ммоль/литр | Десятичное с запятой |

## Особенности извлечения текста
1. **Степени:** `10^12` и `10^9` извлекаются как `1012` и `109`, требуют пост-обработки
2. **Перенос строк:** Единицы измерения часто находятся на отдельной строке, требуют объединения
3. **Разделитель:** Десятичные дроби используют запятую (`,`)

## Проблемы и решения
| Проблема | Решение |
|----------|---------|
| Потеря степеней | Регулярные выражения: `1012` → `10^12` |
| Перенос единиц измерения | Объединение строк по паттернам |
| Мусорные строки (точки-тире) | Фильтрация строк |
