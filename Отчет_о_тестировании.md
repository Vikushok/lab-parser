# Отчёт о тестировании парсера медицинских документов

---

## 1. Объект тестирования

**Программный модуль:** Парсер медицинских показателей из PDF-документов

**Разработанные классы:**
- `PdfReaderTest.java` — извлечение и очистка текста из PDF
- `LabReportParser.java` — парсинг показателей и сохранение в JSON

---

## 2. Цели тестирования

1. Проверить корректность извлечения текста из PDF-документов
2. Проверить качество очистки текста (удаление мусора, объединение строк)
3. Проверить правильность исправления степеней (10^9, 10^12)
4. Проверить точность распознавания медицинских показателей
5. Проверить корректность извлечения числовых значений и единиц измерения
6. Проверить сохранение результатов в JSON-формате

## 7. Обнаруженные проблемы и решения

| Проблема | Решение |
|----------|---------|
| Степени терялись (1012 вместо 10^12) | Регулярные выражения для исправления |
| Единицы измерения на отдельных строках | Объединение строк по паттернам |
| Разорванные строки (LDL-, HDL-C) | Специальная обработка в методе `mergeLines()` |
| FT3 и FT4 извлекались с неверными значениями | Точный поиск по полному названию |
| SARS-CoV-2 IgM не извлекался | Обработка разорванной строки |
